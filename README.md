# MarkdownExport
**Markdown Export for Sublimetext 3/4**

## Usage:
open your Command palette (`Ctrl+Shift+P`) and use `Markdown Export`

## Installation
Clone the repository into your Package Directory
Make sure you name the directory `MarkdownExport`
